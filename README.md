# The Relevance Roadmap

Week 4 microtask tool for **Stay Relevant in an AI World** course.

Participants answer five reflection questions and get a personalized 30-day action plan generated by AI.

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Add your Anthropic API key

Copy the example env file and add your key:

```bash
cp .env.example .env
```

Edit `.env`:

```
ANTHROPIC_API_KEY=sk-ant-...
```

The key stays server-side — it's used by the Vercel serverless function in `api/generate.js` and is never exposed to the browser.

### 3. Run locally

For local development, install the Vercel CLI to run the serverless function locally:

```bash
npm install -g vercel
vercel dev
```

This starts the app at [http://localhost:3000](http://localhost:3000) with the `/api/generate` endpoint working.

## Deploy to Vercel

1. Push to GitHub
2. Import the repo in [vercel.com](https://vercel.com)
3. Add environment variable `ANTHROPIC_API_KEY` in Vercel project settings → Environment Variables
4. Deploy

That's it — the `api/generate.js` file is automatically picked up as a serverless function by Vercel.

## File Upload Support

The tool accepts:
- `.txt` — plain text
- `.doc` / `.docx` — Word documents (via mammoth.js)
- `.pdf` — PDF files (via pdf.js)
- `.pptx` — PowerPoint files (via JSZip)

Participants should number their answers 1–5 or use the question text as headers.

## Tech Stack

- React 18 + Vite
- Anthropic Claude API (Sonnet)
- mammoth.js (DOCX parsing)
- pdf.js (PDF parsing)
- JSZip (PPTX parsing, loaded from CDN)

## Course Context

This tool is the capstone exercise for Week 4 of the Stay Relevant in an AI World course by Lähtijät. It transforms four weeks of reflection into a concrete action plan.
